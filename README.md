# Bootstrap 5 UI Internship Project

## Project Overview
This project was created using HTML5, CSS3, and Bootstrap 5. The goal was to build responsive and modern web pages by remixing Bootstrap components.

## Steps Followed
1. Created project folder and HTML files.
2. Added Bootstrap CDN.
3. Built Home, About, and Contact pages.
4. Added Navbar, Carousel, Cards, and Form.
5. Improved spacing and responsiveness.
6. Tested mobile view.
7. Uploaded to GitHub and deployed using GitHub Pages.

## Tools Used
- Bootstrap 5 Documentation
- GitHub
- VS Code
- ChatGPT (for guidance)

## Challenges Faced
- Understanding responsive grid system.
- Fixing navbar collapse issue.
- Proper spacing and layout alignment.

## Learning Outcome
- Learned Bootstrap components.
- Understood responsive design.
- Improved UI composition skills.
